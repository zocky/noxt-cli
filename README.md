# noxt-cli

A command-line interface for [Noxt](https://npmjs.com/package/noxt-server).

## Installation

```sh
npx noxt-cli create my-app
```

## Usage

```sh
cd my-app
npm run start 
```

or, if you want to run the app with live reload:

```sh
npm run dev
```
